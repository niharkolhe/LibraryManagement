HOW TO RUN:

  - To work on linux systems compile and run linux_main.cpp in Project folder.

    Linux users can also directly execute main.out if present.

  - To work on Windows system compile and run win_main.cpp in Project folder.

    Windows users can directly run main.exe if present.

------------------------------------------------------------------------------------------------------

FEATURES:

  - All data is common to both the systems so it can be used alternatingly without any problems.

WARNING:
